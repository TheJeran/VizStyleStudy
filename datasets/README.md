### raw_mins_ip.csv
This is all of the information received when a participant finishes the last video with the IP field removed 

### users_scored.csv
This is processed data.

**fam#**: is user familiarity with the topic

**form#**: is the visual style for that given topic (LA, LV, AA, AV)

**score#**: is the # of correct answers on multiple choice quiz for that topic

**SA#**: is the # of correct short answers for that topic

### all_scores_format.csv
This is processed from **users_scored.csv** which is grouped to just the visual style

**fam**: is user familiarity with the topic

**form**: is the visual style for that given topic (LA, LV, AA, AV)

**score**: is the # of correct answers on multiple choice quiz for that topic

**SAC**: is the # of correct short answers for that topic

**total** this is **score** + **SAC**
